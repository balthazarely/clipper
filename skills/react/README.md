# React Practice Skills for Clipper

These exercises are designed for this codebase and map directly to existing files in `src/pages` and `src/components`.

## How to Use

1. Start with Skill 1 and go in order.
2. Create a branch per skill so you can review your diff cleanly.
3. Do not jump to polish first. Get behavior working, then refactor.
4. Test each skill manually in the app before moving on.

## Skill Path

1. [Skill 1 - Form Validation and Derived State](./skill-1-form-validation-and-derived-state.md)
2. [Skill 2 - Component Composition and Reuse](./skill-2-component-composition-and-reuse.md)
3. [Skill 3 - Routing, URL State, and Navigation UX](./skill-3-routing-url-state-and-navigation-ux.md)
4. [Skill 4 - Local Persistence and Data Migration Basics](./skill-4-local-persistence-and-migration-basics.md)

## Suggested Time

- Skill 1: 30 to 45 minutes
- Skill 2: 45 to 60 minutes
- Skill 3: 45 to 75 minutes
- Skill 4: 60 to 90 minutes

## Optional Stretch

After each skill, write one short note:

- What broke first?
- What assumptions were wrong?
- What would you change if this needed to scale?
